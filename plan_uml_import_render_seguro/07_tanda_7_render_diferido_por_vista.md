# Tanda 7 — Render diferido por vista interna

## Objetivo

No renderizar todas las vistas internas al mismo tiempo.

## Regla

Solo se renderiza la vista activa. Las demás se renderizan cuando el usuario las selecciona.

## Vistas

- Resumen
- Backend
- Frontend
- Integración API
- Mega vista

## Resultado esperado

La Mega vista no debe abrirse por defecto si el diagrama es grande. Debe requerir confirmación.
