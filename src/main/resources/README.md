# Recursos de la aplicación

Esta carpeta contiene recursos embebidos usados por la aplicación JavaFX.

```txt
css/        estilos del look and feel desktop claro
icons/      iconos de acciones principales
examples/   ejemplos Markdown disponibles desde la app
templates/  plantillas base para crear nuevos modelos
branding/   icono y marca básica del producto
```

## Estado actual

La identidad mínima de R-7 ya está aplicada en `css/app-light.css`. Las siguientes mejoras visuales deben respetar la misma dirección: herramienta desktop clásica, sobria y orientada al modelado conceptual.
