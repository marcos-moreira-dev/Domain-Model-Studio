# Flujo operativo

Objetivo: explicar el trabajo real del negocio sin formalismo excesivo.

## Pasos

1. Recibir solicitud.
2. Revisar datos mínimos.
3. Ejecutar tarea principal.
4. Registrar evidencia.
5. Cerrar atención.

## Riesgos o puntos de atención

- Datos incompletos.
- Doble registro.
- Falta de autorización.
