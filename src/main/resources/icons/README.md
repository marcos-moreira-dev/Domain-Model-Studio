# Iconos internos

Iconos PNG propios, sobrios y aptos para JavaFX.

## Criterios

- La interfaz runtime usa `ImageView` con archivos `.png`.
- No hay dependencia de SVG ni fuentes de glifos para la toolbar.
- Color base `#2F3A4A` y acento `#2F6DB3`.
- Pensados para toolbar, diálogos, documentación y empaquetado local.
