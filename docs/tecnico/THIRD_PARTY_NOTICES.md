# Avisos de terceros

## Estado actual

La aplicación ya no incorpora fuentes de iconos ni recursos Bootstrap embebidos para la toolbar.

La iconografía visible del editor se entrega como PNG locales bajo:

```txt
src/main/resources/icons/
src/main/resources/branding/
```

Los textos visibles de la aplicación siguen usando lenguaje de producto: proyecto, modelo, diagrama, entidad, atributo, relación, cardinalidad, notación, vista, conexión, propiedades, exportación y ayuda.
