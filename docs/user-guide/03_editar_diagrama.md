# 03 - Editar diagrama

## Elementos editables

El MVP contempla edición básica de:

```txt
entidades
atributos
relaciones
etiquetas de cardinalidad
conexiones
puntos intermedios de conexiones
apariencia visual
```

## Movimiento de figuras

Las figuras se pueden mover en el área de diagrama. Al guardar `.dms`, las posiciones corregidas deben conservarse.

## Conexiones con puntos intermedios

Para evitar cruces incómodos, las conexiones pueden tener puntos intermedios.

Uso previsto:

```txt
agregar punto intermedio
mover el punto para esquivar una figura
eliminar puntos innecesarios
guardar esos puntos en el proyecto .dms
```

## Propiedades

El panel **Propiedades** sirve para modificar el elemento seleccionado.

Puede mostrar opciones como:

```txt
nombre visible
posición
tamaño
color de fondo
color de borde
color de texto
fuente
tamaño de fuente
anclas y marcas de conexión
```

## Recomendación

No uses la apariencia visual para cambiar el significado del modelo. Si una relación o cardinalidad está mal, corrige el modelo, no solo el texto visible.
