## Criterio de captura

```txt
- ventana maximizada o con tamaño cómodo;
- ejemplo oficial cargado;
- texto legible;
- sin rutas personales visibles;
- sin consola visible;
- sin mensajes de error accidentales;
- mostrar lenguaje de producto, no lenguaje de desarrollo.
```

## Cuándo hacerlas

Después de I12, cuando el smoke testing real confirme que la aplicación abre, importa, guarda, exporta y muestra el manual correctamente.
