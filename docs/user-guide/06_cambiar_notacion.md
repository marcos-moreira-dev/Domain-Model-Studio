# 06 - Cambiar notación

## Notaciones iniciales

Domain Model Studio contempla dos perfiles iniciales:

```txt
Chen
Pata de gallo
```

## Diferencia conceptual

Chen y pata de gallo no son solo estilos visuales.

### Chen

```txt
entidades como rectángulos
atributos como óvalos externos
relaciones como rombos
cardinalidades como etiquetas
útil para explicar el modelo conceptual
```

### Pata de gallo

```txt
entidades como cajas compactas
atributos dentro de las cajas
relaciones como líneas directas
cardinalidad en los extremos
útil para una vista más compacta
```

## Distribuciones separadas

Cada notación puede tener su propia distribución. Eso es normal.

Un diagrama Chen bien acomodado no necesariamente se traduce a pata de gallo con las mismas posiciones.

## Recomendación

Usa Chen para entender y explicar el dominio. Usa pata de gallo cuando necesites una vista más compacta o más cercana al diseño de datos.
