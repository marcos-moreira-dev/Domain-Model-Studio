# 08 - Guía académica integrada

Domain Model Studio incluye una guía académica dentro de la aplicación, separada de la ayuda operativa del SideDock.

## Cómo abrirla

```txt
Ayuda > Guía académica
```

También puede abrirse con:

```txt
F1
```

## Qué contiene

```txt
Teoría de modelos y diagramas
Notación y reglas de lectura
Cuándo usar cada artefacto
Cuándo no usarlo
Errores comunes
Figuras didácticas
Relación entre teoría y aplicación administrativa
```

## Para qué sirve

La guía académica sirve para estudiar la teoría detrás de cada artefacto: ER, UML, BPMN, C4, matrices, documentos técnicos y artefactos administrativos.

La ayuda operativa de cada herramienta vive en el SideDock del proyecto activo y explica cómo usar esa herramienta concreta.

## Regla de lenguaje

La guía académica habla en términos teóricos y de producto: dominio, artefacto, notación, diagrama, documento, matriz, wireframe, ejemplo, plantilla, propiedades, vista y exportación.
