# 04 - Guardar y abrir `.dms`

## Qué es `.dms`

`.dms` es el archivo editable de Domain Model Studio.

No es lo mismo que el Markdown.

```txt
Markdown = entrada o intercambio con IA
.dms     = proyecto editable con tipo, datos y ajustes
PNG/SVG/PDF/Markdown = salidas para revisión o entrega
```

## Qué debe guardar

Un `.dms` debe conservar:

```txt
metadatos del proyecto
tipo de diagrama o documento
datos semánticos principales
ruta del Markdown fuente cuando exista
ajustes visuales o documentales disponibles
```

## Flujo recomendado

1. Importar Markdown.
2. Ajustar la salida central.
3. Guardar `.dms`.
4. Volver a abrir `.dms` para continuar.
5. Exportar cuando necesites revisar o entregar.

## Regla importante

Si ya corregiste manualmente un proyecto, trabaja sobre `.dms`. No vuelvas a crear todo desde Markdown sin necesidad, porque podrías perder ajustes realizados en la aplicación.
