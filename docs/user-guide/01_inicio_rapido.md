# 01 - Inicio rápido

## Objetivo

Convertir un Markdown oficial o generado con IA en una salida editable y exportable.

## Pasos generales

1. Abrir Domain Model Studio.
2. Elegir **Importar Markdown** o crear un proyecto nuevo del tipo necesario.
3. Usar una plantilla o ejemplo oficial de `examples/markdown/`.
4. Revisar la salida central:
   - diagrama;
   - mapa;
   - matriz;
   - wireframe;
   - documento.
5. Ajustar manualmente lo mínimo necesario.
6. Guardar el proyecto como `.dms`.
7. Exportar el formato aplicable.

## Archivos de ejemplo

La carpeta `examples/markdown/diagramas/` contiene ejemplos oficiales por tipo.

La carpeta `examples/markdown/plantillas/` contiene plantillas base para pedirle a una IA Markdown equivalente del cliente.

## Recomendación práctica

Para proyectos reales, empieza con el Markdown. La edición manual sirve para corregir presentación y detalles, no para reemplazar el levantamiento de información.
