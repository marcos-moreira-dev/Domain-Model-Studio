# 09 - Glosario de producto

Este glosario explica palabras que aparecen en la aplicación y en la guía.

## Términos principales

```txt
Proyecto              archivo de trabajo editable en Domain Model Studio
Markdown              entrada estructurada o salida actualizada del proyecto
Plantilla             base oficial para escribir un Markdown nuevo
Ejemplo               caso oficial listo para revisar o adaptar
Modelo                conjunto de elementos y relaciones de un dominio
Diagrama              representación visual de un modelo o proceso
Documento             salida textual/tabular profesional, como diccionario de datos
Matriz                vista cruzada, por ejemplo rol × permiso
Wireframe             boceto visual de una pantalla
Entidad               concepto principal del dominio
Atributo              dato o característica de una entidad o relación
Relación              vínculo conceptual entre elementos
Cardinalidad          cantidad permitida o esperada en una relación
Notación              forma visual de representar un modelo
Chen                  notación conceptual con entidades, atributos y relaciones separados
Pata de gallo         notación compacta con entidades y relaciones directas
Propiedades           panel para revisar y ajustar el elemento seleccionado
Conexión              línea visual entre elementos del diagrama
Exportación           salida como PNG, SVG, PDF o Markdown según corresponda
Guía académica       referencia teórica integrada abierta desde el Menú Ayuda
Ayuda operativa       guía práctica del SideDock sobre la herramienta activa
```

## Idea central

Domain Model Studio ayuda a pasar de información escrita a una salida revisable, editable y exportable.
