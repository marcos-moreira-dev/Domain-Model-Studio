# Flujo operativo

Objetivo: atender un ticket técnico desde el registro hasta el cierre.

## Pasos

1. Recibir solicitud del cliente.
2. Registrar síntoma y prioridad.
3. Revisar equipo o servicio afectado.
4. Proponer solución o derivar.
5. Registrar evidencia del trabajo.
6. Cerrar ticket con conformidad.

## Puntos de atención

- Confirmar datos de contacto.
- Evitar cerrar sin evidencia.
