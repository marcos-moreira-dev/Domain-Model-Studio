# Plantillas oficiales Markdown

Cada plantilla representa el esqueleto mínimo de un tipo visible del catálogo. Se usan para pedir a una IA un Markdown equivalente del cliente sin inventar formatos.

## Regla

- Mantener `diagram_type` como ID oficial.
- Mantener `importable` coherente con la capacidad real del tipo.
- Usar los ejemplos UENS cuando haga falta una referencia más completa.
- No convertir diccionario y roles en canvas libre: diccionario es documento; roles/permisos es matriz.
- Usar `free-graph` solo cuando el objetivo sea una red flexible de nodos y relaciones.
